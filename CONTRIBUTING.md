# Contributing

Thank you for helping improve this profile repository.

## Contribution Priorities
- clarity and readability
- accessibility and inclusive language
- accurate links and up-to-date references
- SEO-friendly, useful, and truthful content

## Pull Request Rules
1. Keep changes focused and minimal.
2. Explain the reason and expected impact.
3. Avoid promotional spam or low-value edits.
4. Preserve brand voice and professionalism.

## Content Quality Standards
- Use descriptive headings.
- Prefer practical examples.
- Avoid exaggerated claims.
- Keep external links relevant and trusted.
