# On-Page SEO Checklist

- Use one clear H1 per page
- Keep title under ~60 characters
- Keep meta description around 140-160 characters
- Include target keyword in first paragraph
- Add descriptive alt text for all images
- Use internal links between related pages
- Use external links only to trusted sources
- Keep paragraphs concise and scannable
- Add FAQ section for long-tail discovery
- Update stale content regularly
