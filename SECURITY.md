# Security Policy

## Supported Scope
This repository hosts profile and public-facing content. Security concerns may include:
- exposed tokens/secrets
- malicious links
- impersonation risks
- harmful or unauthorized content

## Reporting a Vulnerability
Please report security concerns privately to: `security@saieshwar.xyz`.

Include:
- affected file or URL
- reproduction details
- potential impact
- suggested mitigation (optional)

## Response Commitment
- Initial response target: within 72 hours
- Status updates: provided as investigation progresses
- Public disclosure: coordinated after remediation

## Safe Harbor
Good-faith security research and responsible disclosure are welcomed.
